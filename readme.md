# 介绍
从0到1用webpack5搭建一个vue2的cli
# 安装
* ```npm init```
# 运行
* ```npm run dev``` ：开发环境运行
* ```npm run build``` : 生产环境运行
# 详情
* ```build```文件夹有三个webpack的配置文件，根目录的```webpack.config.js```用于初次构建webpack项目使用
